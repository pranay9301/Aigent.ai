# Competitive Gap Analysis — Aigent.ai vs AI Workflow Competitors

## Scope
This document maps current Aigent.ai capabilities against major automation/AI-workflow platforms and identifies high-value gaps to close next.

## Current Aigent.ai strengths
- One workspace combining AI chat, code editor, preview, and deploy.
- Built-in roles: PM, DEV, DES, QA, marketing, debug, explain, UI improve, security, finance.
- Gemini-backed orchestration plus neural caching for repeated prompts.
- Invoice generation + Razorpay billing with live key validation.

## Competitors evaluated
- Zapier
- Make.com
- n8n
- Gumloop
- Power Automate / Workato / Tray.ai

## Key gaps
1. Missing workflow marketplace / templates beyond starter archetypes.
2. No human-in-the-loop automation builder; current deploy is manual code generation.
3. Limited integration catalog outside Razorpay and basic Firebase auth.
4. No timeline/gantt or delivery tracking for AI-generated builds.
5. No export/report pack for AI finance tool beyond role-level copy.

## Proposed first-mover features
- AI Workflow Runner: let users define reusable automation recipes with resume/replay.
- Role Gallery Switcher: matrix view of agents per project, plus agent run logs.
- Vendor Integration Pack: Gmail, Notion, Linear, GitHub, Vercel, Resend wrappers.
- Deployment Health Radar: one-page health + billing + deployment status.
